## 一、用户注册/登录（P0）

### US-001 用户注册

**作为** 新用户，**我想要** 使用邮箱和密码注册账号，**以便** 能够使用中集智历的各项功能。

**验收标准：**
- [ ] 提供邮箱、密码、确认密码输入框
- [ ] 邮箱格式验证，确保是有效的邮箱格式
- [ ] 密码强度要求：至少8位，包含字母和数字
- [ ] 密码和确认密码必须一致
- [ ] 注册时必须选择并回答一个安全问题（用于密码找回）
- [ ] 安全问题选项：母亲姓名、出生城市、第一所学校名称、最喜爱的电影
- [ ] 邮箱不能与已注册用户重复
- [ ] 注册成功后自动登录并跳转到仪表盘

---

### US-002 用户登录

**作为** 注册用户，**我想要** 使用邮箱和密码登录系统，**以便** 访问我的项目和任务。

**验收标准：**
- [ ] 提供邮箱和密码输入框
- [ ] 登录失败时显示明确的错误提示（邮箱不存在/密码错误）
- [ ] 登录成功后跳转到仪表盘
- [ ] 支持"记住我"功能，延长登录状态有效期
- [ ] 密码输入框支持显示/隐藏密码切换

---

### US-003 密码找回

**作为** 忘记密码的用户，**我想要** 通过安全问题重置密码，**以便** 重新获得账号访问权限。

**验收标准：**
- [ ] 登录页面提供"忘记密码"入口
- [ ] 输入邮箱后显示该用户设置的安全问题
- [ ] 正确回答安全问题后可以设置新密码
- [ ] 新密码需满足强度要求
- [ ] 密码重置成功后需要重新登录
