# 中集智历 前端设计规范

> 版本：1.0.1
> 更新日期：2026年3月1日
> 变更说明：基于现有中集智历.HTML设计，确保UI一致性

---

## 一、设计原则

### 1.1 核心原则

| 原则 | 说明 |
|------|------|
| **保持一致** | 新系统UI与现有HTML完全一致 |
| **复用样式** | 直接复用现有Tailwind CSS类名 |
| **扩展功能** | 在现有设计基础上扩展新功能 |
| **响应式设计** | 保持移动端适配效果 |
| **简洁美观** | 维持现有清爽的视觉风格 |

### 1.2 设计来源

本设计规范基于 [中集智历.HTML](../中集智历.HTML) 提取，确保迁移到Vue 3后视觉效果完全一致。

## 十一、迁移对照表

### 11.1 React → Vue 3 对照

| React | Vue 3 | 说明 |
|-------|-------|------|
| useState | ref / reactive | 状态管理 |
| useEffect | onMounted / watch | 副作用 |
| useMemo | computed | 计算属性 |
| props | defineProps | 组件属性 |
| onClick | @click | 事件绑定 |
| className | :class / class | 类名绑定 |
| {condition && <Comp/>} | v-if | 条件渲染 |
| {list.map(item => ...)} | v-for | 列表渲染 |

### 11.2 样式复用示例

**现有HTML (React)**：
```jsx
<button className="w-full flex items-center gap-3 px-4 py-3 rounded-lg bg-blue-600 text-white">
  {icon}
  <span className="font-medium text-sm">{label}</span>
</button>
```

**迁移后 (Vue 3)**：
```vue
<button class="w-full flex items-center gap-3 px-4 py-3 rounded-lg bg-blue-600 text-white">
  <component :is="icon" />
  <span class="font-medium text-sm">{{ label }}</span>
</button>
```

**Tailwind类名完全一致！**
