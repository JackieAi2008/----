# 中集智历 需求规格说明书

> 版本：1.0.0
> 更新日期：2026年3月1日

---

## 一、产品概述

### 1.1 产品定位

中集智历是一款**轻量级工作协同平台**，介于团队日历工具和项目管理系统之间，强调"日程可视化"的同时具备基本的任务跟踪能力。

### 1.2 目标用户

| 项目 | 描述 |
|------|------|
| 团队规模 | 10-50人 |
| 使用场景 | 公网SaaS服务 |
| 部署方式 | 自建服务器（Mac mini M4） |

### 1.3 核心功能

1. **工作日历**：以日历形式展示任务和日程
2. **项目管理**：支持创建/加入多个工作项目
3. **任务协同**：任务指派、状态跟踪、评论讨论
4. **数据统计**：工作报表、进度分析、工作总结

### 2.2 登录方式

| 登录方式 | 是否支持 | 说明 |

用户注册时需设置一个安全问题，忘记密码时通过回答安全问题验证身份：

1. 您的母亲姓名是什么？
2. 您的出生城市是哪里？
3. 您的第一所学校名称是什么？
4. 您最喜爱的电影是什么？

## 三、项目管理

### 3.1 项目的定义

在中集智历中，**项目代表具体工作事项**，例如：
- 2026年度党建工作
- 五一活动筹备
- 半年度考核

### 3.2 项目属性

| 属性 | 类型 | 说明 |
|------|------|------|
| 项目名称 | 文本 | 必填，最多50字 |
| 项目描述 | 文本 | 选填，最多500字 |
| 项目封面 | 图片 | 选填，用于视觉识别 |
| 可见性 | 枚举 | 公开/私密 |
| 创建时间 | 日期 | 自动记录 |
| 创建者 | 用户 | 自动记录，即项目负责人 |

### 3.3 项目可见性

| 类型 | 描述 |
|------|------|
| 公开项目 | 可被搜索到，用户可申请加入 |
| 私密项目 | 不可搜索，只能通过邀请加入 |

### 3.4 成员加入方式

- **恢复机制**：30天内负责人可恢复项目
- **永久删除**：30天后系统自动清除数据

## 四、任务管理

### 4.1 任务属性

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 标题 | 文本 | ✅ | 任务名称，最多100字 |
| 任务描述 | 文本 | ❌ | 详细说明，最多2000字 |
| 开始时间 | 日期时间 | ❌ | 任务开始日期和时间 |
| 截止时间 | 日期时间 | ✅ | 任务截止日期和时间（精确到分钟） |
| 类别 | 枚举 | ❌ | 系统默认或项目自定义 |
| 负责人 | 用户 | ✅ | 主要负责人（仅1人） |
| 协作人员 | 用户列表 | ❌ | 协助人员（可多人） |
| 优先级 | 枚举 | ❌ | 高/中/低 |
| 状态 | 枚举 | ✅ | 未开始/进行中/已完成/已取消 |
| 交付成果 | 文本 | ❌ | 需要提交的产出物 |
| 标签 | 文本列表 | ❌ | 自定义标签 |
| 提醒 | 枚举 | ❌ | 提前多久提醒 |
| 重复 | 枚举 | ❌ | 不重复/每天/每周/每月/每年 |
| 附件 | 文件列表 | ❌ | 相关文件 |
| 创建者 | 用户 | 自动 | 任务创建人 |
| 创建时间 | 日期时间 | 自动 | 任务创建时间 |
| 更新时间 | 日期时间 | 自动 | 最后修改时间 |

### 4.2 任务类别

**系统默认类别**：
- 党建
- 群团
- 行政
- 节假日

**项目自定义类别**：
- 项目负责人可在项目设置中添加自定义类别
- 自定义类别仅在当前项目生效

### 4.3 任务指派规则

- 每个任务只有**一个主要负责人**
- 负责人必须是项目成员
- 协作人员可以是多人
- 任务会显示在负责人和协作人员的日历上

### 4.4 任务提醒

| 提醒时机 | 说明 |
|----------|------|
| 提前15分钟 | 任务即将开始 |
| 提前1小时 | 任务即将开始 |
| 提前1天 | 任务即将到期 |
| 提前3天 | 任务即将到期 |

## 五、日历与视图

### 5.1 视图类型

| 视图 | 说明 | 优先级 |
|------|------|--------|
| 月视图 | 默认视图，显示整月任务 | 第一期 |
| 周视图 | 显示一周任务，可看到具体时间 | 第一期 |
| 日视图 | 显示单日详细日程 | 第一期 |
| 列表视图 | 按时间顺序列出任务 | 第二期 |
| 甘特图视图 | 显示任务时间线和依赖关系 | 第二期 |

### 5.2 默认页面布局

用户登录后默认显示：
- **左侧/上方**：仪表盘概览
  - 今日待办
  - 逾期任务
  - 本周/本月概览
  - 项目快速切换
- **右侧/下方**：日历视图
  - 可切换月/周/日视图
  - 显示用户相关的所有任务

### 5.3 跨项目任务展示

用户参与多个项目时，在日历上通过以下方式区分：
- **颜色区分**：不同项目用不同颜色标记
- **标签显示**：任务上显示所属项目名称
- **筛选切换**：可筛选只看某个项目的任务

### 5.4 查看他人日程

项目成员可以查看同项目其他成员的日程：
- **筛选查看**：通过筛选框选择查看某人的日程
- **并排对比**：同时勾选多人，日历叠加显示
- **成员主页**：点击成员头像进入该成员的日程页面

## 十五、术语表

| 术语 | 定义 |
|------|------|
| 项目 | 具体工作事项的集合，如"2026年度党建工作" |
| 任务 | 项目中的具体工作项，有明确的负责人和时间 |
| 项目负责人 | 项目的创建者或被移交者，拥有项目最高权限 |
| 普通成员 | 项目参与者，可以创建和管理任务 |
| 公开项目 | 可被搜索到，用户可申请加入的项目 |
| 私密项目 | 不可搜索，只能通过邀请加入的项目 |
| PWA | 渐进式Web应用，可添加到桌面，支持离线 |
| 软删除 | 数据标记为删除但保留一段时间，可恢复 |
